# Heterogeneous Runtime Analysis for Quantum and Classical ML

Profiled training runtime of **Variational Quantum Classifiers** (Qiskit statevector simulator) vs classical **SVMs** (scikit-learn) across 16–784 MNIST features, measuring wall-clock time and memory usage using Python's `time.perf_counter()` and `tracemalloc` to analyze O(n) scaling behavior on CPU-based simulation.

Collected systematic performance measurements across 100+ training runs, identifying non-linear scaling patterns in quantum simulation (R²=0.11–0.54) vs predictable O(n²) classical behavior.

## Project Structure

```
├── datasets/               # Place MNIST CSV files here (mnist_train.csv, mnist_test.csv)
├── results/                # Benchmark CSVs (auto-generated by benchmark_runner.py)
├── src/
│   ├── svm_classifier.py   # SVM classification pipeline (single-run)
│   ├── vqc_classifier.py   # VQC classification pipeline (single-run)
│   ├── benchmark_runner.py  # Sweep 16–784 features, profile runtime & memory
│   └── plot_metrics.py     # Accuracy + runtime scaling plots
├── requirements.txt
└── README.md
```

## Setup

```bash
pip install -r requirements.txt
```

## Dataset

Download the MNIST dataset in CSV format and place the following files in the `datasets/` directory:

- `mnist_train.csv`
- `mnist_test.csv`

Each CSV should have a `label` column followed by 784 pixel columns.

## Usage

```bash
# Run single SVM classifier (with perf_counter + tracemalloc profiling)
python src/svm_classifier.py

# Run single VQC classifier (with perf_counter + tracemalloc profiling)
python src/vqc_classifier.py

# Sweep 16–784 features and benchmark both models (saves CSVs to results/)
python src/benchmark_runner.py              # both SVM + VQC
python src/benchmark_runner.py --model svm  # SVM only
python src/benchmark_runner.py --model vqc  # VQC only

# Generate accuracy + runtime scaling plots
python src/plot_metrics.py
```
